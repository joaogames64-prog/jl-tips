# JL Tips — Apostas Esportivas

App mobile-first de gestão profissional de apostas esportivas.

## Funcionalidades

- 📊 **Dashboard** — ROI, Yield, Win Rate, gráfico de evolução
- ➕ **Registrar Apostas** — Simples, Múltipla, Back/Lay, 60+ mercados
- 📋 **Lista de Apostas** — Filtros por esporte, resultado, casa, data
- 📈 **Analytics** — +20 métricas, gráficos, breakdown por categoria
- 💰 **Multi-Bankroll** — Múltiplas bancas, depósitos, saques
- 🧮 **Calculadoras** — Kelly Criterion, Surebet, Dutching, Odds
- 📅 **Filtros por Data** — Hoje, ontem, 7d, 30d, 90d, período personalizado
- 💾 **IndexedDB** — Banco de dados local persistente
- 📱 **PWA** — Instale no celular como app nativo

## Stack

- HTML5, CSS3 (vanilla), JavaScript ES6+
- Chart.js para gráficos
- IndexedDB para persistência
- Service Worker para offline/PWA

## Como usar

```bash
# Servir localmente
python3 -m http.server 3000

# Acessar
http://localhost:3000
```
